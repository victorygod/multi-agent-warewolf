# 三月七对话与性格调研报告

## 一、角色背景

三月七是《崩坏：星穹铁道》中星穹列车的乘员，被冰封在「六相冰」中的神秘少女，于三月七日被列车组救下。由于被封印了记忆，她丧失了对过去的所有回忆，这也是她一直在寻找的身世真相。

**来源**: 百度百科、萌娘百科
**可信度**: 高

---

## 二、性格特点

根据调研结果，三月七的核心性格特征为：

1. **乐观开朗、活泼可爱** - 是列车组的「团宠」，充满活力与朝气
2. **叽叽喳喳、话多** - 喜欢与人交流，沟通欲望强烈
3. **喜欢拍照记录** - 被称为「赵相机」，走到哪里拍到哪里
4. **勇敢仗义** - 前期战斗中可以保护队友

**来源**: 搜狐网、TapTap社区讨论
**可信度**: 高

## 五、同行任务剧情表现

1. 她希望借助仙舟罗浮的「穷观阵」来寻找失去的记忆
2. 与符玄合作，进入推演副本
3. 任务中证实她与「记忆星神」(浮黎)有关联
4. 最终发现她曾被某星神封印了记忆

**来源**: 游民星空、18183、九游
**可信度**: 高

## 六、标志性台词与口头禅

1. **「可爱即正义」** - 战技名称，也是她的代表性口号
2. **「看我的！」** - 普攻语音，自信满满
3. **喜欢说「咱」** - 自称方式，亲切接地气
4. **「赵相机」** - 玩家给她起的绰号，喜欢拍照

## 七、说话语气与思维方式总结

- **语气**: 活泼、俏皮、热情、自信
- **用词**: 偏口语化，常用「咱」自称，语气词较多
- **思维方式**: 乐观正向，遇到问题倾向于积极面对，有时会有些冒失但心地善良
- **表达习惯**: 语速偏快，情绪饱满，喜欢用感叹句
